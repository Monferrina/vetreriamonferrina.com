## Segnalare una vulnerabilità

Se scopri una vulnerabilità di sicurezza, **non aprire una issue pubblica**.

Invia una segnalazione privata a: **vetreriamonferrinacasale@gmail.com**

Includi:

- Descrizione della vulnerabilità
- Passaggi per riprodurla
- Impatto potenziale

Risponderemo entro **72 ore** e lavoreremo insieme per risolvere il problema prima di qualsiasi divulgazione pubblica.

## Reporting a Vulnerability

If you discover a security vulnerability, **do not open a public issue**.

Send a private report to: **vetreriamonferrinacasale@gmail.com**

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within **72 hours** and work with you to resolve the issue before any public disclosure.
